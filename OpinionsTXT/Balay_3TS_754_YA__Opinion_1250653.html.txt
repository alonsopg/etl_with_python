=========
Cajón Dosificador
===============
El cajón dosificador de productos de lavado y aditivo esta divido en tres cámaras.
Cámara I, detergente para el prelavado o almidonado.
Cámara II, detergente para lavado principal, descalcificador del agua, ablandadores, blanqueadores o sal quitamanchas.
Camara III, (que se encuentra en medio de las dos anteriores), para el suavizante de ropa. Advierten que si el suavizante es muy espeso a de diluirse con un poco de agua, para no obstruir el sifón).


====================
El mando selector de programas
==========================Programas básicos: 
 Resistentes algodon frio. 40, 60 º C (Algodón o lino). 
 Resistentes algodon. 60º C extra. (Para algodón o lino, pero que esten bastante sucios o con manchas)
   
 Resistentes algodón 90º C. Este programa es el que más energía consume por lavar las prendas a tan alta temperatura y está indicado para algodón y lino. 
Y algo que me parece excesivo aunque sea bueno a la larga es que dice que para proteger los conductos de aguas residuales, antes de efectuar el vaciado mezcla el agua de lavado caliente con aproximadamente 7 litros de agua fría. Asi que aparte de consumir energía gasta muchisima agua, yo no he utilizado este programa, porque no tengo prendas tan sucias como para necesitar este tipo de lavado. 

* Sinteticos frío, 40 o 60º C. Para prendas de fácil cuidado con el algodón, lino y tejidos sintéticos y mixtos. Este es que se usa como el conocido programa corto. :-D* Mix 30º C, este programa esta optimizado para tejidos poco sucios de algodón y prendas de fácil cuidado. Se pueden lavar juntos tipos distintos de prendas. Adecuado también para lavar tejidos nuevos antes de usarlos por primera vez. Y su tiempo de lavado es también corto. 


* Delicado frío. Este esta dedicado para prendas muy delicadas, seda, satén, tejidos sintéticos y mixtos (hasta las cortinas) y lana pues lava estos tejidos como si los lavaras a mano. 
* Proceso de aclarado con centrifugado final para aclarar ropa lavada a mano o para almidonar. 


* Centrifugado adicional, como dice el nombre es para centrifugar cuando no estas satisfecha y quieres quitarle más agua a las prendas lavadas. Aunque con el centrifugado del lavado para mi es más que suficiente, pues sale muy bien la ropa, no sale chorreando al reves, a veces depende del tejido, sale casi seco :-D. 
Además de este mando tiene unas teclas adicionales que cada una tiene una función.


*Redu. Centrif. Pulsando esta tecla, el pilote de la función se ilumina. En el programa de lavado para la ropa blanca y de color de tejidos resistentes, el centrifugado se ejecuta a menor velocidad. 
*Prelava. Para tejidos resistentes de algodón o lino, que presenten mucha suciedad. La temperatura del prelavado es de 30º C. 


* Flot. Para tejidos delicados y cortinas. Todos los programas se pueden efectuar sin el centrifugado final. La ropa se queda en el tambor lleno de agua de aclarado. Se enciende el piloto luminoso. Y antes de retirar la ropa deberemos ejecutar el programa de ";Centrifugado adicional";. 
Y por ultimo el botón de incio, para que la lavadora comience a lavar :-D

=====
Mi experiencia
===========


Pues como veis tiene muchos programas y muchos acciones adicionales para hacer del lavado de prendas algo más personal y al gusto, es decir que como se puede lavar casi todo y con su programa, no tienes porque temer a que se estropee la ropa, es super delicado y muy acertado cada uno de los programas. 
Respecto al ruido y las vibraciones de momento es bastante silenciosa, solo se le escucha cuando está centrifugando pero no es para tanto, tambien sera porque esta fuera en un cuartito. Y las vibraciones si la ajustas bien al suelo con las patitas ajustables que tiene no se movera ni un pelo. 
Y respecto al precio a mi me costó 383 euros. 


La verdad que no se como serán el resto de lavadoras, yo la que tenía antes era de todo menos buena y fácil de usar, por eso os cuento lo fácil y lo bien que se lava la ropa con esta lavadora. :-D
